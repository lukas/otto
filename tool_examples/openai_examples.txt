#### Prompt:

openai(prompt="[prompt]") asks openai a question. 

#### Prompt Examples:

### User: Ask openai if whales are mammals
### Assistant: openai(question="Are whales mammals?")

### User: Ask gpt if a leopard can swim
### Assistant: openai(prompt="Can a leopard swim?")
